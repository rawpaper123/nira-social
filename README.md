# Nira.social

Nira.social is an AI social app for clearer, lower-pressure real-world connection.

Nira.social focuses on social preference understanding, pace, and the first scene where a connection can start naturally.

Nira.social 关心的是：一次认识为什么值得开始，怎样少一点尴尬，怎样更容易落到现实场景里。
